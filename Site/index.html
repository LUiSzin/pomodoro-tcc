Oque é o pomodoro timer

Pomofocus é um temporizador pomodoro personalizável que funciona no desktop e navegador móvel.
O objetivo deste aplicativo é ajudá-lo a se concentrar em qualquer tarefa em que você esteja
trabalhando, como estudo, escrita ou codificação. Este app é inspirado na Técnica Pomodoro,
que é um método de gerenciamento de tempo desenvolvido por Francesco Cirillo.


Para mais informações:

O que é a Técnica Pomodoro?

A Técnica Pomodoro é criada por Francesco Cirillo para uma forma mais produtiva de trabalhar
e estudar. A técnica usa um temporizador para dividir o trabalho em intervalos, tradicionalmente
com 25 minutos de duração, separados por intervalos curtos. Cada intervalo é conhecido como pomodoro,
da palavra italiana para "tomate", depois do temporizador de cozinha em forma de tomate que Cirillo
usou como estudante universitário.

Como usar a técnica Pomodoro?

01- Adicione tarefas para trabalhar hoje
02- Definir pomodoros de estimativa (1 = 25min de trabalho) para cada tarefa
03- Selecione uma tarefa para trabalhar
04- Inicie o temporizador e foque na tarefa por 25 minutos
05- Faça uma pausa por 5 minutos quando o alarme tocar, volte ao passo 04
